# 🚗 Ilumi Carros

Sistema web desenvolvido para a **Ilumi** com foco em **reserva, controle e gestão de veículos corporativos**, permitindo organização de agendamentos, usuários e histórico completo de utilização da frota.

---

## 📌 Sobre o Projeto

O **Ilumi Carros** foi criado para modernizar o processo interno de utilização de veículos da empresa, substituindo controles manuais por uma plataforma digital rápida, segura e organizada.

O sistema permite que colaboradores solicitem veículos e que a portaria/administração acompanhem toda a operação em tempo real.

---

## ⚙️ Funcionalidades

### 🔐 Autenticação de Usuários

* Login seguro via Supabase
* Controle de acesso por perfil

### 👥 Perfis do Sistema

* Portaria
* Funcionário
* Administrador

### 📅 Reservas de Veículos

* Solicitação de veículos
* Escolha de data e horário
* Bloqueio automático de horários ocupados
* Validação de disponibilidade

### 🚘 Gestão da Frota

* Cadastro de veículos
* Controle de disponibilidade
* Status dos veículos

### 📝 Histórico de Uso

* Registro de reservas realizadas
* Controle de saída e retorno
* Histórico operacional

### 📊 Administração

* Criação e edição de reservas
* Exclusão de registros
* Gestão de usuários

---
